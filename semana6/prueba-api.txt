Prueba de la ruta POST /registro

Herramienta utilizada: Thunder Client (dentro de Codespaces)

Solicitud enviada a: http://localhost:3000/registro
Método: POST
Cuerpo (JSON):
{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Respuesta recibida del servidor:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

La respuesta confirma que el servidor recibió correctamente los datos enviados y los devuelve como confirmación.
