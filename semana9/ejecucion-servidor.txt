Cuando ejecuto node server.js, el servidor se inicia y la terminal muestra el mensaje "Servidor ejecutándose en puerto 3000". El servidor queda a la espera de solicitudes HTTP en ese puerto.

Al visitar la ruta /estado desde el navegador (http://localhost:3000/estado), el servidor responde con un objeto JSON:
{
  "estado": "Servidor funcionando",
  "servicio": "API comunitaria"
}

Esto confirma que el servidor está activo y puede devolver información estructurada.
